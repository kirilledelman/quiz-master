# Quiz Master

This project exists to showcase use of **React**, **React Router**, and **Redux store**. 

**/dist** contains files ready to be deployed to a web server,\
**/api** contains the server portion of the site, to be run using [Node.js](https://nodejs.org)

Uses [React](https://react.dev) as the main framework,\
[React Router](https://reactrouter.com) for routing URLs, loaders and form actions,\
[Redux Toolkit](https://react-redux.js.org/) for application state,\
[LowDB](https://github.com/typicode/lowdb) for simple JSON based database server-side.

Compiled using [Vite](https://vite.dev) tool.
